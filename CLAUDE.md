# DailyLearn

学習アプリのフロントエンド試作。`index.html` がエントリで、`screens/*.jsx` を Babel-in-browser で読み込む構成。

## ドメイン用語(UI 文言の統一基準)

UI に表示する日本語は以下の語に揃える。新規画面・コピー追加時もこれに従う。

- **コース** … 全体(30日間の学習プログラム)。「プラン」は使わない。
- **レッスン** … 1日分の学習コンテンツ。
- **AIアシスタント** … チャット相手の AI(略す場合は「アシスタント」、フル名は「AIアシスタント」)。
- **無料プラン / 有料プラン** … 課金プランを指す場合のみ「プラン」を使う(コースの意味では使わない)。

コース期間は **30日** に統一済み。`screens/roadmap.jsx` の `Array.from({ length: 30 })` と `screens/home.jsx` のチップ `DAY xx / 30` が単一ソース。
