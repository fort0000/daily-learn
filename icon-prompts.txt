================================================================================
  DailyLearn アイコン画像生成プロンプト集
================================================================================

このファイルは、アプリで使用している画像アセットを「アプリ独自のテイスト」に
差し替えるための画像生成プロンプトと、各アイコンの使用箇所をまとめたものです。

推奨ツール:
  - ChatGPT (gpt-image-1) … シリーズの統一感が出しやすい
  - Recraft               … SVG 出力できるのでアイコン用途と相性◎
  - Ideogram              … 数字・文字を含めたい場合に強い

作業フロー:
  1. 「マスタースタイル」をすべてのプロンプトの先頭に貼る
  2. 各アイコンのプロンプトで個別の被写体を指示
  3. 1024×1024 を生成 → sips / squoosh で各サイズへダウンサイズ
  4. public/ 配下の同名ファイルを上書き(コード側の参照パスは変更不要)

================================================================================
  マスタースタイル(全プロンプトの先頭に付与)
================================================================================

Flat vector illustration, friendly modern Japanese learning app icon style.
Strong rounded shapes, thick navy outline (#0F172A, approx 6% of canvas width).
Subtle warm gradient fill. Soft drop shadow offset down 4-6px, no blur,
color #C8431A at 30% opacity (hanko / Japanese stamp feel).
Palette strictly: warm orange #FF7A45 / fire #F97316 / mint #22C55E /
cream #FFF5E6 / navy #0F172A / yellow accent #FACC15.
No text, no letters, no Japanese characters. Centered subject with ~12% padding.
CRITICAL — TRANSPARENT BACKGROUND: Output MUST be a PNG with an alpha channel.
The entire background MUST be fully transparent (alpha = 0). Do NOT add a
white, cream, or any solid-color background fill. Only the subject is
visible; everything outside the subject silhouette is empty / alpha 0.
1024x1024 square.
Avoid: photorealism, 3D render, glossy material, gradient mesh, busy detail,
white background, cream background, any solid background plate.

目的:
  プロフィール画面のバッジセクション。現在は絵文字(📚🔥💎🌟🏆)を直接描画

================================================================================
  生成後の後処理メモ
================================================================================

ダウンサイズ(macOS の sips を使う例):

  # アプリアイコン
  sips -z 512 512 master.png --out public/icon-512.png
  sips -z 192 192 master.png --out public/icon-192.png
  sips -z 180 180 master.png --out public/apple-touch-icon.png

  # ファビコン(別途生成したシンプル版から)
  sips -z 32 32 favicon-master.png --out public/favicon-32.png
  sips -z 16 16 favicon-master.png --out public/favicon-16.png

  # バッジ
  sips -z 512 512 badge-first-master.png --out public/badges/badge-first.png
  (以下同様)

  # 完了モーダル
  sips -z 256 256 trophy-master.png --out public/complete-trophy.png
  sips -z 64 64 book-master.png    --out public/complete-progress-book.png

確認:
  - 透過 PNG として書き出されているか(背景が白塗りになっていないか)
    確認コマンド:
      sips -g hasAlpha path/to/image.png
    → "hasAlpha: yes" でなければ NG(白背景のまま)。
    その場合は再生成 or 画像編集ソフトで白を抜く。
  - 16×16 でつぶれていないか、ブラウザタブで実際に確認
  - ダークモード対応が必要なら、シルエットが薄背景でも視認できるか
